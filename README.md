# C-Humidity_Sensor
# -> Определить влажность почвы с помощью датчика влажности HL-69

# -> Компненты для сборки:
# - Макетная плата
# - Arduino Uno
# - Датчик влажности HL-69
# - Перемычки
# - Светодиоды 
# - Резисторы 220кОм
# 
# -> Логика работы:
# Датчик влажности измеряет сопротивление между контактами и отдает уровень влажности.
# В цикле работы платы считываются аналоговые данные с датчика, подгоняются под истинные отдаваемые значения (в моем случае максимально низкое зн-е - 350, высокое - 1021) и преобразуются в значение от 0 до 6ти.
# В зависимости от значения от датчика с помощью цикла поочередно загораются светодиоды.
# Для дебага программы был использован класс Serial, с помощью которого можно вывести логи в консоль.
